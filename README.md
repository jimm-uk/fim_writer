# fim_writer
Fill in the Middle Markdown Editor. Use this tool to have Gen-AI generated text recommendations for writing activities such as blog writing. Super lightweight editor written with HTML only to make calls to LLMs for text recommendations. Any OpenAI compatible API can be used giving model choice.
